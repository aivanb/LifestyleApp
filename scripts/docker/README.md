# Docker helper scripts

Cross-platform commands are in the root [README.md](../../README.md#docker). Use these scripts if you prefer a single entrypoint.

## Bash (Linux / macOS / Git Bash)

- `dev-up.sh` — build and start the development stack
- `dev-down.sh` — stop development stack
- `dev-clean.sh` — stop and remove volumes (wipes MySQL data)
- `prod-up.sh` — build and start production stack (detached)
- `prod-down.sh` — stop production stack
- `prod-clean.sh` — stop production stack and remove volumes

## PowerShell (Windows)

Same names with `.ps1` extension.
